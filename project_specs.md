# Project Specs

## What the app does
A local SEO marketing site used as a base for a YouTube tutorial. Starts as a clean landing page; features will be added during the tutorial series.

## Who uses it
Visitors interested in local SEO (tutorial audience).

## Tech stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Supabase — not yet, will add when needed
- Hosting: localhost for now, Vercel later

## Pages
- `/` — Landing page (hero, value prop, "coming soon" section)

## Data models
None yet.

## Third-party services
None yet.

## What "done" looks like (this task)
- `npm run dev` boots cleanly on `http://localhost:3000`
- Landing page renders with premium, modern styling (no emoji, no generic gradients)
- `npm run build` passes
- File structure matches CLAUDE.md
- `.env.local` placeholder exists; `.gitignore` excludes it
